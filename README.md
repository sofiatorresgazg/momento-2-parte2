Este proyecto realiza la carga, limpieza y analisis de ventas y productos utilizando Python y la librerias 

## Estructura del Proyecto
* `analisis.py`: Script principal que ejecuta el anaisis y muestra resultados.
* `funciones.py`: Contiene las funciones de limpieza, carga y estandarizacion
* `data/raw/`: Carpeta con los archivos CSV originales.
* `requirements.txt`: Lista de libreria ncesarias para el proyecto.

## Requisitos
Para ejecutar este proyecto, es necesario tener instalado Python y las dependencias listadas en el archivo requirements.
